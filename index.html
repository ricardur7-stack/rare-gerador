Ferramentas Avançadas (API)

Você pode usar esta API para resetar o vínculo de um dispositivo com a chave de forma programática. Use uma requisição **PUT** para o endpoint abaixo, substituindo `KEY_ID` pelo ID da chave.

Endpoint:

PUT https://rare-android-565a4-default-rtdb.firebaseio.com/keys/KEY_ID.json

Corpo da requisição (JSON):

{
  "deviceId": null
}
